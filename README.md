Class to Canteen (C2C) 🍔🏫
Class to Canteen is a dedicated food-ordering platform that beautifully streamlines campus dining operations. Students can pre-order meals directly from their classrooms, eliminating long queues and optimizing lunch breaks. The system features real-time order tracking, digital payments, and smarter inventory management to reduce overall food waste.
=====================================
🚀 Features
Classroom Pre-Ordering: Browse the daily canteen menu and place orders in advance.

Queue Management: Receive estimated preparation times and notifications when food is ready for pickup.

Digital Payments: Secure and seamless checkout process integrated directly into the platform.

Vendor Dashboard: A dedicated interface for canteen staff to manage incoming orders, update item availability, and track daily sales.

Cross-Platform Experience: Optimized for both mobile and desktop use to serve students on the go.
